508 /********************* SPI Slave FSM *******************************
509 **cmd\status PWROFF PWRON ConfigRead(CR) ConfigWrite(CW)
510 ** Poweroff N Y Abnormal Abnormal
511 ** Poweron Y N N N
512 **ConfigRead N Y Abnormal Abnormal
513 ** Read N N Y Abnormal
514 **ConfigWrite N Y Abnormal Abnormal
515 ** Write N N Abnormal Y
516 
517 Note: N: invalid operation.
518  Y: the operation completed successfully.
519  Abnormal: an error occurred during the operation.
520 *******************************************************************/
